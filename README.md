# VBA_script

Gồm một số đoạn script cho các tác vụ của Excel như tách nội dung câu, kết hợp nhiều cells.

**Thư mục**
1. Images: gồm một số ảnh hướng dẫn sử dụng
2. Modules: gồm một số script function
